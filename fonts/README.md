# Fonts

## CascadiaSarasa-Regular.ttf

Custom merged font combining Cascadia Code and Sarasa Term SC Nerd.
Install by copying to `~/.local/share/fonts/merged/` and running `fc-cache -fv`.

## Additional fonts (download separately)

- **Cascadia Code** (all variants): https://github.com/microsoft/cascadia-code/releases
  Install to `~/.local/share/fonts/cascadia/`

- **Sarasa Term SC Nerd** (all variants): https://github.com/laishulu/Sarasa-Mono-SC-Nerd/releases
  Install to `~/.local/share/fonts/sarasa/`

After installing all fonts, run:
```sh
fc-cache -fv
```
